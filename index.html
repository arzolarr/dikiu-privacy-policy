Política de Privacidad de Dikiu Client

La aplicación Dikiu Client está diseñada para permitir a los usuarios visualizar la ubicación de su dispositivo móvil en tiempo real, así como realizar pruebas con diferentes plataformas de monitoreo mediante la modificación de la dirección IP. Esta aplicación no recopila, almacena ni comparte datos personales del usuario.

Datos de ubicación

La aplicación Dikiu Client accede a la ubicación del dispositivo únicamente con el fin de mostrar su posición en el mapa dentro de la plataforma de monitoreo. Esta información no se comparte con terceros ni se almacena fuera del dispositivo. El acceso a la ubicación se realiza solo cuando el usuario lo permite expresamente.

Acceso a datos

Dikiu Client no accede a contactos, fotos, archivos, ni ningún otro dato personal del dispositivo. El único permiso solicitado es el de ubicación, y su uso está limitado a la funcionalidad descrita anteriormente.

Consentimiento del usuario

El acceso a la ubicación se solicita mediante el sistema de permisos de Android. Antes de que se active este permiso, se muestra una explicación clara dentro de la aplicación sobre el propósito del acceso. El usuario puede aceptar o rechazar este permiso en cualquier momento desde la configuración del dispositivo.

Modificaciones

Esta política puede actualizarse en el futuro para reflejar cambios en la funcionalidad de la aplicación o en los requisitos legales. Se recomienda revisar esta página periódicamente.
